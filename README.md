# 📊 Hindi Authorship Attribution Dataset

A curated dataset for multi-author Hindi authorship attribution, designed to support research in Natural Language Processing (NLP) for low-resource languages.

---

## 🔹 Overview

- **Dataset Name:** hindi_authorship_attribution_dataset  
- **Language:** Hindi (Devanagari script)  
- **Task Type:** Multi-class classification (Authorship Attribution)  
- **Domain:** Literary Text (Stories & Poems)  

---

## 🔹 Dataset Structure

Each row represents a paired textual sample:

| Column | Description |
|--------|------------|
| Author | Ground truth label (author name) |
| Story  | Prose text sample |
| Poem   | Poetry text sample |
